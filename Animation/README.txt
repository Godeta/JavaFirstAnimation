Voici un premier projet, on y trouve une animation simple : 

- un gif pour le fond
- une boucle chargeant différentes images pour le personnage de Durarara
- une boucle se déplaçant dans l'image pour le personnage que j'ai créé avec un créateur de sprite gratuit en ligne (:

+ détection touches

Le jar doit être dans le même dossier que le dossier Images !!!

Possibilité de stocker les images, sons, vidéos, bibliothèques... sur un autre serveur et de ne garder que le code, avec git large file storage !

La version actuelle demande d'avoir la version de Java adéquate ou de recompiler le code, si l'on veut un executable portable 
il faudrait fournir le JRE avec !
